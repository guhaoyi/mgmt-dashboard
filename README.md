# 雪梨经营管理看板

> 基于飞书电子表格数据的交互式经营分析看板，纯前端单文件部署。

## 在线访问

**https://guhaoyi.github.io/mgmt-dashboard/**

## 项目概况

| 项目 | 说明 |
|------|------|
| 数据源 | 飞书电子表格「雪梨营业收入利润情况表」 |
| 数据范围 | 2025年1月 ~ 2026年3月（15个月） |

| 部署方式 | GitHub Pages（纯静态，零后端） |
| 文件 | 单文件 `index.html`（~60KB） |

## 交互特性

| 特性 | 说明 |
|------|------|
| 时间筛选 | 起止月份下拉 + 快捷按钮（全部/近6月/近12月/2025/2026） |
| Crosshair | 鼠标悬浮时竖向虚线吸附到数据点 |
| Tooltip | 所有图表 `mode:index`，悬浮显示同月所有系列完整金额 |
| 数据标签 | 关键图表直接在图上显示数字/百分比，无需悬浮 |
| 环形图标注 | 直接显示百分比占比 |
| 表格 | 首列冻结、表头冻结、负数红色、合计行高亮 |
| 响应式 | 移动端自适应布局 |

- 表格：9px表头，10px数据，首列+表头冻结

### Chart.js 配置模板
```javascript
// 全局默认
Chart.defaults.color = '#8896b0';
Chart.defaults.borderColor = 'rgba(30,35,51,0.5)';
Chart.defaults.font.family = "'JetBrains Mono','Noto Sans SC',sans-serif";
Chart.defaults.font.size = 9;

// 必须配置项
interaction: { mode: 'index', intersect: false }  // 所有图表
plugins: { datalabels: { display: false } }        // 默认关闭，按需开启

// Crosshair 插件（竖向虚线）
const crosshairPlugin = {
  id: 'crosshair',
  afterDraw(chart) {
    if (chart.tooltip?._active?.length) {
      const ctx = chart.ctx;
      const x = chart.tooltip._active[0].element.x;
      const top = chart.chartArea.top;
      const bottom = chart.chartArea.bottom;
      ctx.save();
      ctx.setLineDash([4, 4]);
      ctx.strokeStyle = 'rgba(139,150,180,0.5)';
      ctx.lineWidth = 1;
      ctx.beginPath();
      ctx.moveTo(x, top);
      ctx.lineTo(x, bottom);
      ctx.stroke();
      ctx.restore();
    }
  }
};
```

## 数据结构

数据以 JS 对象 `D` 内嵌在 HTML 中，每个字段是长度15的数组（index 0=2025年1月，14=2026年3月）。

完整字段清单见 `data.json`。

### 数据层级
```
前端收入（10个平台）→ 前端成本（6类）→ 前端毛利
后端收入（5个平台）→ 后端成本（4类）→ 后端毛利
后端毛利 - 人力成本（8类）- 运营管理费（6+类）= 后端EBIT
前端毛利 + 后端EBIT ± 财务/折旧/营业外 = 税前利润
税前利润 - 总经办 - 税金 = 净利润
```

## 如何新增看板 / 合并到总看板

### 方案A：新增独立看板（推荐起步）
1. 复制 `index.html` 为模板
2. 替换 `D` 对象中的数据
3. 修改 Tab 和图表配置
4. 保持相同的 CSS 变量和组件规范

### 方案B：合并为总看板
1. 将各业务看板作为顶层 Tab 或路由
2. 共享 CSS 变量、字体、图表配置
3. 数据层改为按业务模块加载
4. 建议结构：
```
总看板/
├── index.html          # 入口 + 路由
├── css/style.css       # 共享样式
├── js/
│   ├── config.js       # Chart.js 全局配置 + 插件
│   ├── helpers.js      # fmt/kpi/table 等工具函数
│   ├── dashboard-mgmt.js    # 经营管理看板
│   ├── dashboard-sales.js   # 销售看板（未来）
│   └── dashboard-xxx.js     # 其他看板
└── data/
    ├── mgmt.json
    ├── sales.json
    └── xxx.json
```
