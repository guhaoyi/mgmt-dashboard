### 数据结构（三层利润表逻辑）

```
前端收入（支付口径）
  - 各平台明细（抖音/淘宝/小红书/拼多多/视频号/京东/B站/快手/有赞/线下）
- 前端成本（广告投放/平台费用/印刷/直播/短信/快递包材）
= 前端毛利

后端收入（支付口径）
  - 各平台明细（小鹅通/淘宝/有赞/香港店铺/线下）
- 后端成本（平台费用/印刷/APP使用费/快递包材）
= 后端毛利

后端毛利
- 人力成本（固定工资/奖金/劳务费/加班/社保/公积金/辞退补偿/福利/招聘）
- 运营管理费（房租/咨询/办公/差旅/软件/视频制作/通讯/设备租赁等20+项）
= 后端息税前利润（EBIT）

前端毛利 + 后端EBIT = 前后端合计
± 财务费用 ± 理财收益 - 折旧摊销 ± 营业外收支
= 税前利润
- 总经办费用（管理团队工资 + 其他）
- 税金
= 净利润
```

### V2 图表重新规划

**核心原则：每个数据找最适合的图表形式，不堆叠**

#### Tab1 总览
1. **KPI 卡片 × 6**: 总收入/总成本/前端毛利/后端毛利/税前利润/净利润（带环比箭头）
2. **收入趋势折线图**: 前端收入 vs 后端收入，双线对比，Y轴统一
3. **净利润柱状图**: 正负双色柱状图，直观看盈亏月份
4. **利润瀑布图**: 从收入到净利润的完整拆解（横向瀑布）
5. **月度利润明细表**: 关键行项目，数字带颜色

#### Tab2 前端分析
1. **KPI 卡片**: 前端收入/成本/毛利/毛利率/抖音占比
2. **平台收入堆叠柱状图**: 看各平台月度贡献
3. **平台占比饼图**: 2025全年
4. **成本结构堆叠面积图**: 看成本构成变化
5. **毛利柱状图**: 正负双色，看哪个月亏

#### Tab3 后端分析
1. **KPI 卡片**: 后端收入/成本/毛利/毛利率/小鹅通占比
2. **平台收入趋势**: 小鹅通为主，其他平台辅助线
3. **毛利趋势**: 柱状图 + 毛利率折线叠加

#### Tab4 费用分析
1. **KPI 卡片**: 人力成本/运营管理费/总经办/人均成本
2. **人力成本堆叠柱状图**: 固定工资/奖金/社保/公积金等
3. **运营管理费 TOP10 横向柱状图**: 按金额排序
4. **费用占收入比趋势**: 折线图

#### Tab5 利润分析
1. **KPI 卡片**: 前后端合计/税前利润/净利润/净利率
2. **利润层级对比**: 前端毛利 vs 后端EBIT vs 税前利润 vs 净利润
3. **完整利润表**: 带条件格式的数据表
